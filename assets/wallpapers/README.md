# Windows Nerd Themes - Assets

Place your background images here. The installer will copy them to Windows Terminal's roaming state folder.

## Star Wars Theme Images

The following images are used by the `star-wars` theme:

- `wallpaper_deathstar.jpg` — Ubuntu (WSL) tab background
- `wallpaper_xwing.jpg` — PowerShell tab background
- `venator.png` — PowerShell Admin tab background
- `wallpaper_falcon.jpg` — CMD tab background
- `desktop_star_wars.jpg` — Desktop wallpaper

## Adding Your Own Images

1. Place images in this folder
2. Reference them by filename in your theme JSON under `background_image`
3. Alternatively, provide a `background_image_url` — the installer downloads it automatically

## Recommended Dimensions

- Terminal backgrounds: 1920×1080 or higher (dark/subtle images work best)
- Desktop wallpaper: Match your display resolution
